Streamline — UML Component Diagram

React Native Expo · Feature-Based Architecture · Supabase Backend

features/camera

  • • 6 hooks: barcode, capture, permissions, location, building detector, accuracy
  • • 2 components: CameraView, BarcodeViewfinder
  • • On-device TFLite model via frame processor worklet
  • • GPS ref-snapshot for zero-latency coordinate capture

features/field-test

  • • Context provider wraps entire app at RootLayout
  • • GroundTruthModal for 1-tap annotation (confusion matrix)
  • • FieldTestBanner indicates active test mode
  • • Extended payload with timing + model diagnostics

utils/ (Service Layer)

  • • api.ts + field-test-api.ts → Supabase Storage + PostgreSQL
  • • exif-builder.ts: GPS → EXIF metadata construction
  • • exif-processor.ts: EXIF writing + gallery save
  • • hasher.ts: SHA-256 image fingerprint for tamper detection
  • • supabase_client.ts: Singleton with AsyncStorage auth

types/ (Shared Domain)

  • • EXIFMetadata: GPS, timestamps, camera settings
  • • LocationCoordinate: lat, lon, accuracy, altitude, speed, heading
  • • Used across features/camera, utils, and app layers
  • • Single source of truth for cross-cutting data shapes
